# Changelog

Журнал изменений проекта «Тренажер-самоучитель по картам Таро».

Формат ведения:
- новые изменения сначала добавляются в раздел `Unreleased`;
- после коммита запись переносится в раздел с датой и hash коммита;
- описания пишутся простым языком: что изменилось для пользователя и какие страницы затронуты.

## Unreleased

- В галерею Таро добавлена викторина «Угадай значение карты»: показывается карта и четыре варианта смысла.
- В раздел «36 карт» добавлена такая же викторина по значениям обычной колоды.
- В обеих викторинах добавлены подсказки: 50/50 и текстовый ключ по группе, масти, рангу или ключевым словам.
- Исправлено скрытие неверных вариантов в подсказке 50/50.
- Крупные разделы галерей и страницы 36 карт переведены в сворачиваемые блоки для удобного мобильного чтения.
- Сворачиваемые панели распространены на основные разделы всех страниц проекта, включая главный тренажер, руководство и тренажер раскладов.
- В полноэкранном просмотре карт добавлено переключение по тапу: карта показывает описание, описание возвращает карту; свайпы между картами сохраняются.
- Временный каталог `.chrome-test/` удален из рабочей папки.
