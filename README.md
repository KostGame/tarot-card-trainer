# Тренажер-самоучитель по картам Таро

Статическое мобильное HTML-приложение на русском языке для самостоятельного изучения карт Таро.

## Цель проекта

Создать простой автономный тренажер, который помогает изучать значения карт, повторять материал и постепенно запоминать систему Таро через короткие упражнения.

## Текущий статус

Создана базовая стартовая страница проекта.

## Технические принципы

- главный файл сайта: `index.html`;
- без backend;
- без npm, сборки и внешних библиотек;
- весь основной код приложения хранится в `index.html`;
- деплой через GitHub Pages из ветки `main`, папка `/root`.

## GitHub Pages

Рекомендуемая настройка:

1. Settings
2. Pages
3. Source: Deploy from a branch
4. Branch: `main`
5. Folder: `/root`
